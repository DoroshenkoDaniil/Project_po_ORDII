Project\_po\_ORDII

Анализ данных сна, здоровья и профессий людей

Данный проект представляет собой комплексное исследование набора данных, содержащего информацию о взаимосвязи сна, здоровья и профессиональной деятельности людей. Анализ выполнен в рамках учебного проекта и включает полный цикл обработки данных, их визуализацию и публикацию результатов.



Стек технологий

Язык программирования: Python 3



Библиотеки для анализа: Pandas, NumPy



Визуализация: Matplotlib, Seaborn



Инструменты: Google Colab, GitHub Desktop, Yandex DataLens



Этапы исследования

Обзор данных: Оценка структуры датасета, проверка типов данных и корректность загрузки.



Предобработка:



Поиск и обработка пропущенных значений.



Выявление и удаление дубликатов.



Приведение столбцов к нужным типам данных.



Стандартизация и кодирование категориальных признаков (например, профессий).



Статистический анализ:



Расчет описательных статистик (среднее, медиана, стандартное отклонение) для показателей сна и здоровья.



Корреляционный анализ между продолжительностью сна, качеством отдыха, физическими показателями здоровья и уровнем стресса.



Анализ различий между профессиональными группами.



Визуализация: Построение распределений, корреляционных карт и сравнительных диаграмм (гистограммы, boxplot, heatmap, scatter plot) для наглядного отображения закономерностей.



Группировка и агрегация: Анализ данных с использованием groupby и pivot\_table для выявления средних показателей сна и здоровья в разрезе различных профессий.



Структура репозитория

Sleep\_health\_and\_lifestyle\_dataset.csv — исходный набор данных.



Doroshenko\_project.ipynb — рабочий блокнот с кодом и выводами.



README.md — документация проекта.



 Основные выводы

Качество данных: Проверка показала, что данные в целом хорошо структурированы. Пропуски были найдены в нескольких столбцах и корректно обработаны, что позволило провести основной анализ без серьезных потерь информации.



Выявленные закономерности:



Обнаружена статистически значимая отрицательная корреляция между продолжительностью сна и уровнем стресса, а также между качеством сна и частотой физических недомоганий.



Анализ показал существенные различия в средних показателях сна и здоровья между представителями разных профессиональных сфер. Например, профессии, связанные с высоким уровнем ответственности и ненормированным графиком, демонстрируют худшие средние значения качества сна и более высокий уровень стресса.



Статистические выбросы: Анализ выявил несколько статистических выбросов среди показателей физической активности и качества сна. Вывод: эти индивиды демонстрируют аномально высокие показатели здоровья даже при нестандартном режиме сна, что может указывать на индивидуальные физиологические особенности или иные компенсирующие факторы образа жизни.
